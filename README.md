# CS3243_Sudoku
Sudoku: CS3243 Artificial Intellgence Project

## Algorithm
Apply csp to solve puzzles, and compare the efficiency of different heuristics:
- Variable selecting:
  - Most constrained variable
  - Degree heuristic
- Value selecting:
  - Least constraining value
- Domain reducing:
  - Forward checking

## Combination (FC for all)
- FC
- MCV
- DH
- LCV
- MCV + DH
- MCV + LCV
- DH + LCV
- MCV + DH + LCV
